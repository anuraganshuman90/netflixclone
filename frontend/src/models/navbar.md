# Navbar

## Responsibility

Provides primary navigation throughout the application.

Allows users to move between pages and access global application features.

---

## Owns

- Navigation links
- Logo / Brand
- Profile entry
- Theme toggle (future)
- Search shortcut (future)

---

## Used By

- App

---

## Inputs

- Navigation items
- Current active page (later)

---

## Outputs

- Navigation events
- Route changes

---

## Should Know

- Available pages
- Navigation structure
- Current active page

---

## Should NEVER Know

- Content
- ContentCollection
- ContentService
- API endpoints
- Database
- Authentication implementation
- Recommendation logic
